# Asset Rules

이미지, CSS, JS, 폰트 같은 정적 파일을 관리할 때 적용합니다.

## 관리 기준

- 사용하지 않는 큰 파일은 추가하지 않습니다.
- 이미지 파일명은 용도를 알 수 있게 정합니다.
- CSS와 JS는 프로젝트의 기존 폴더 구조를 따릅니다.
- 외부 CDN을 추가할 때는 수업 환경에서 인터넷 접근이 가능한지 확인합니다.

## Git 기준

올릴 수 있는 파일:

- 직접 만든 이미지
- 수업에서 사용하는 CSS, JS
- 화면 구현에 필요한 정적 파일

주의할 파일:

- 용량이 큰 원본 이미지
- 임시 캡처
- 다운로드 받은 압축 파일
