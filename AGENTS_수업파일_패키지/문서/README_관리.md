# README 관리

저장소 README와 각 폴더 README를 수정할 때 적용합니다.

## 루트 README

루트 `README.md`에는 저장소 전체 구조와 Git 관리 기준만 둡니다.

넣을 내용:

- 저장소 목적
- 큰 폴더 구조
- Git에 올릴 파일과 제외할 파일
- 에이전트 문서 위치

넣지 않을 내용:

- 개별 Java 코드 풀이
- 긴 디버깅 과정
- 특정 날짜의 상세 작업 기록
- 임시 실행 결과

## 폴더 README

각 폴더 README에는 그 폴더에서 바로 필요한 정보만 둡니다.

- `Java2`는 Java/Eclipse 기준
- `WEB_CLOUD`는 JSP, Servlet, Tomcat 기준
- `front_수업파일`은 HTML, CSS, JavaScript 기준
- `AIAgent`는 AI 에이전트 문서와 사용 규칙 기준
