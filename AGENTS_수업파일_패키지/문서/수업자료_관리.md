# 수업자료 관리

`AcornFileMain` 아래 수업자료를 정리할 때 적용합니다.

## 주요 경로

```text
AcornFileMain/Java2/수업자료/
AcornFileMain/Java2/workspace/
AcornFileMain/WEB_CLOUD/수업소스/
AcornFileMain/front_수업파일/
AcornFileMain/SQLD/
AcornFileMain/AIAgent/
```

## 관리 기준

- PDF, 강의 txt, 예제 소스는 원본을 보존합니다.
- 직접 푼 실습 코드는 Git에 올릴 수 있습니다.
- Eclipse `.metadata/`, `bin/`, `*.class`는 Git에 올리지 않습니다.
- 로그, lock, 임시 빌드 결과는 Git에 올리지 않습니다.
- 같은 이름의 실습 파일이 많으면 날짜, Day 번호, 패키지 기준으로 구분합니다.

## 정리할 때 확인할 것

- Day 번호가 맞는지 확인합니다.
- Eclipse 패키지명과 폴더 경로가 맞는지 확인합니다.
- 수업자료와 직접 작성한 풀이 파일을 섞어 설명하지 않습니다.
- 대량 이동이나 삭제가 필요하면 먼저 사용자에게 확인합니다.
