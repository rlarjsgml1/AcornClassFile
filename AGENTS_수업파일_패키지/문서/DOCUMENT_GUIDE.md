# 문서 폴더

코드가 아닌 설명, 정리, README, 수업자료 관리 기준을 모아두는 폴더입니다.

## 언제 보는가

- README를 수정할 때
- 수업자료, PDF, txt, 정리 문서를 정리할 때
- 코드 설명 문서나 사용방법 문서를 만들 때
- 개발로그와 디버깅로그에 남길 내용을 판단할 때

## 문서 목록

```text
문서_작성규칙.md
수업자료_관리.md
README_관리.md
```

## 기본 원칙

- 문서는 코드 동작을 과장하지 않습니다.
- 확인한 사실과 추정 내용을 구분합니다.
- 경로, 파일명, 실행 방법은 실제 파일을 확인한 뒤 적습니다.
- 수업 원본 자료는 가능한 한 보존하고, 정리는 별도 파일로 남깁니다.
