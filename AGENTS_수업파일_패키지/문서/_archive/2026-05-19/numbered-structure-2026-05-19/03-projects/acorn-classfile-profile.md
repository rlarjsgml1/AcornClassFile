# AcornClassFile Profile

`C:\AcornFile` 저장소 전용 규칙입니다.

## 저장소 성격

이 저장소는 에이콘 수업 자료, 실습 코드, 학습 정리 파일을 모아두는 개인 학습 저장소입니다.

## 주요 경로

```text
C:\AcornFile
C:\AcornFile\AcornFileMain\Java2
C:\AcornFile\AcornFileMain\WEB_CLOUD
C:\AcornFile\AcornFileMain\front_수업파일
C:\AcornFile\AcornFileMain\SQLD
C:\AcornFile\AcornFileMain\AIAgent
C:\AcornFile\AGENTS_수업파일_패키지
```

## 작업 기준

- Java2는 Eclipse workspace 기준으로 봅니다.
- WEB_CLOUD는 JSP, Servlet, Tomcat 수업 프로젝트 기준으로 봅니다.
- 프론트 수업 파일은 HTML, CSS, JavaScript 기준으로 봅니다.
- 수업 원본 자료는 보존하고, 정리는 별도 문서로 남깁니다.

## Git 제외 기준

Git에 올리지 않을 파일:

- `.metadata/`
- `bin/`
- `*.class`
- `*.lock`
- `*.log`
- `.codex-build/`

## SourceTree 기준

- Stage All 전에 변경 파일을 확인합니다.
- 수업 소스와 IDE 자동 생성 파일을 구분합니다.
- push, reset, merge, rebase는 사용자 확인 후 진행합니다.
