AI 코딩 에이전트 지침을 실무형 구조로 정리한 패키지입니다.

## 바로 볼 파일

- `AGENTS.md`: 에이전트가 가장 먼저 읽는 공통 진입 문서
- `00-overview/package-map.md`: 전체 폴더 구조 설명
- `00-overview/usage-guide.md`: 적용 방법
- `03-projects/acorn-classfile-profile.md`: `C:\AcornFile` 저장소 전용 규칙

## 사용 기준

- 새 프로젝트에 적용할 때는 `AGENTS.md`와 필요한 하위 폴더를 함께 둡니다.
- 기존 `docs`, `code`, `backend` 등 옛 구조는 `99-archive/legacy-2026-05-19`에 보관했습니다.
- 작업 중 새 규칙이 생기면 바로 `AGENTS.md`에 넣지 말고 알맞은 하위 문서에 추가합니다.
