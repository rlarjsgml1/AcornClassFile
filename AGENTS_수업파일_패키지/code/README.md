# code 폴더

실제 코드 작업 기준을 언어와 실행 환경별로 나눈 폴더입니다.

## 먼저 볼 문서

```text
JAVA_ECLIPSE.md       Java2, Eclipse, 일반 Java 수업 코드
JAVA_WEB.md           JSP, Servlet, Tomcat, WEB_CLOUD
ANDROID_KOTLIN.md     Android Studio, Kotlin, Gradle
JAVASCRIPT.md         JavaScript, 브라우저, Node.js
SQL_DATABASE.md       SQL, DB, DAO와 연결되는 쿼리
```

## 공통 확인

- 실제 프로젝트 경로를 먼저 확인합니다.
- 패키지명과 폴더 경로가 맞는지 확인합니다.
- 기존 코드 스타일을 유지합니다.
- 컴파일 또는 실행 확인을 했는지 분명히 기록합니다.
- 불필요한 리팩터링은 하지 않습니다.
