# UX Checklist

사용자 흐름을 점검할 때 사용합니다.

## 확인 항목

- [ ] 첫 화면에서 무엇을 해야 하는지 알 수 있는가?
- [ ] 클릭 가능한 요소와 읽기 전용 요소가 구분되는가?
- [ ] 오류가 났을 때 사용자가 다음 행동을 알 수 있는가?
- [ ] 모바일 화면에서 글자와 버튼이 겹치지 않는가?
- [ ] 불필요한 설명 문구가 화면을 막지 않는가?
- [ ] 반복 작업이 너무 많은 클릭을 요구하지 않는가?

## 웹 수업 화면 기준

- JSP 화면은 서버에서 내려준 데이터와 사용자가 입력하는 데이터를 구분합니다.
- 폼 제출 후 성공, 실패, 빈 값 상태를 확인합니다.
- Ajax 화면은 요청 중, 성공, 실패 상태를 구분합니다.
