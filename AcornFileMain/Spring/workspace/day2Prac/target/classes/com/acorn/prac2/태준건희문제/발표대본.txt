고양이 집사 적합도 계산기 발표대본

안녕하세요.
저희는 객체 생성과 주입 예제로 고양이 집사 적합도 계산기를 만들었습니다.

이 프로그램은 놀이 시간, 인내심, 청소 습관, 양육 예산, 책임감이라는 5가지 항목을 입력받습니다.
각 항목은 10점 만점이고, 전체 점수는 100점 기준으로 계산됩니다.

먼저 Calculator 인터페이스를 만들었습니다.
이 인터페이스는 calcScore라는 계산 메서드의 규칙을 정합니다.
그리고 CatButlerCalculator 클래스가 이 인터페이스를 구현해서 실제 점수 계산을 담당합니다.

Program 클래스는 계산기를 직접 만들지 않고 Calculator 타입으로 주입받아서 사용합니다.
그래서 계산 방식이 바뀌어도 Program 코드는 크게 바꾸지 않아도 됩니다.
이 부분이 이번 과제의 핵심인 객체 생성과 주입입니다.

첫 번째 실행 방식은 ProgramMain입니다.
여기서는 Spring을 사용하지 않고 Main에서 Program 객체와 CatButlerCalculator 객체를 직접 만들고,
setCalculator 메서드로 직접 주입했습니다.

두 번째 실행 방식은 SpringProgramMain입니다.
여기서는 setting.xml 파일을 읽어서 Spring 컨테이너가 객체를 만들고 주입합니다.
XML 안에서 calculator bean과 program bean을 등록하고, property 태그로 calculator를 program에 넣었습니다.

세 번째 실행 방식은 Controller입니다.
CatButlerController에서 /catButler URL 요청을 받습니다.
요청 파라미터로 5가지 점수를 받고, Program을 이용해서 점수와 결과 문구를 계산합니다.
그 다음 Model에 값을 담아서 catButlerView.jsp 화면에 출력합니다.

예를 들어 URL에 playTime=8, patience=9, cleaning=7, budget=8, responsibility=10을 넣으면
총점이 계산되고 "좋은 집사 후보" 같은 결과가 화면에 표시됩니다.

정리하면, 이 예제는 같은 계산 기능을
Main에서 직접 주입하는 방식,
Spring XML로 주입하는 방식,
그리고 Controller에서 URL 요청으로 실행하는 방식
세 가지로 보여 주는 실습입니다.

이상입니다.
