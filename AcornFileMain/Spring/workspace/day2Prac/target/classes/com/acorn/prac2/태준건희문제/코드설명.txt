고양이 집사 적합도 계산기 코드 설명

1. 주제
5순위 추천 주제인 "고양이 집사 적합도 계산기"를 객체 생성과 주입 방식으로 구현했다.
사용자의 놀이 시간, 인내심, 청소 습관, 양육 예산, 책임감 점수를 받아 총점을 계산한다.

2. 파일 역할
Calculator.java
- 계산 기능의 규칙을 정하는 인터페이스이다.
- calcScore 메서드는 5가지 항목을 받아 점수를 반환한다.

CatButlerCalculator.java
- Calculator 인터페이스를 구현한 실제 계산기 클래스이다.
- 각 항목은 10점 만점으로 보고, 2를 곱해서 총점 100점 기준으로 계산한다.

Program.java
- 계산기를 직접 사용하지 않고 Calculator 타입으로 주입받아 사용한다.
- 생성자 주입, 세터 주입, Spring 자동 주입을 모두 보여 줄 수 있다.
- 점수 계산과 결과 등급 문구 생성을 담당한다.

ProgramMain.java
- Spring을 사용하지 않고 Main에서 직접 객체를 만들고 주입하는 예제이다.
- Program을 만들고, CatButlerCalculator를 setCalculator로 직접 넣는다.

SpringProgramMain.java
- setting.xml을 읽어서 Spring 컨테이너로 실행하는 예제이다.
- Spring이 Program과 Calculator 객체를 만들고, Calculator를 Program에 주입한다.

setting.xml
- XML 방식으로 bean을 등록하는 지시서이다.
- calculator bean과 program bean을 만들고, property 태그로 세터 주입을 한다.

CatButlerController.java
- 브라우저 URL 요청으로 실행하는 Spring MVC Controller이다.
- /catButler 요청을 받으면 점수를 계산하고 JSP에 보여 줄 값을 Model에 담는다.

catButlerView.jsp
- Controller가 전달한 값을 화면에 출력하는 JSP 파일이다.

3. 실행 방법
직접 객체 생성 실행:
- ProgramMain.java 실행

Spring XML 실행:
- SpringProgramMain.java 실행

브라우저 실행:
- http://localhost:8080/day2Prac/catButler?playTime=8&patience=9&cleaning=7&budget=8&responsibility=10

4. 전체 흐름
브라우저 요청이 들어오면 CatButlerController가 값을 받는다.
Controller는 Program에게 계산을 맡긴다.
Program은 주입받은 CatButlerCalculator를 사용해서 점수를 계산한다.
계산 결과와 등급 문구는 Model에 담겨 catButlerView.jsp 화면에 출력된다.
