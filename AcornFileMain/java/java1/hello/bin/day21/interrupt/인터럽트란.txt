
	 interrupt();
	 신호보내기, 쿡찌르기
	 "너에게 신호를 보냈으니 알아서 처리해"
	
	 interrupt() 메서드는 
	 
	 자바에서 쓰레드에 인터럽트를 걸어 특정 작업을 중지하거나 다른 작업을 수행하도록 신호를 보내는 데 사용됩니다.
	 
	 ##알아두기 ##
	 interrupt()는 강제로 쓰레드를 중지시키는 것이 아니라
	 쓰레드에 인터럽트 상태를 설정하여 그 쓰레드가 현재 수행 중인 작업을 적절히 중단하도록  하는것이다.
	 
	 
	 
	 인터럽트 보내기    (보내는 쪽)  ->  	   (받는 쪽)   인터럽트를 받은 객체에서 필요한 작업을 하는 것이다.
	 
	 
	 #인터럽트 오해하지 말기 !!
	 #단순히 인터럽트를 보낸다고  그 대상이 종료되거나 하지 않는다 단지 인터럽트 신호만 보낸것이다 
	 
	 
	 
	
	# 인터럽트 확인 하기 
	
	인터럽트 확인 메서드(2개 제공 한 개만 기억하자 !!)

    isInterrupted()   :인터럽트가 걸렸는지 확인하기 (true, false)
    Thread.interrupted() :인터럽트가 걸렸는지 확인하기 (true, false) : 다시 초기화함  =>알아만 두자 
    
    
    ✔️ 특징
    특정 스레드의 인터럽트 상태 확인
    ❗ 상태를 초기화하지 않음 (유지됨)
    
	✔️ 예제
	Thread.currentThread().interrupt();
	System.out.println(Thread.currentThread().isInterrupted()); // true
	System.out.println(Thread.currentThread().isInterrupted()); // true
    계속 true 유지

2️   
    Thread.interrupted()
    Thread.interrupted();
✔️ 특징
    현재 스레드의 인터럽트 상태 확인
    확인하면서 상태를 false로 초기화
    
✔️ 예제
    Thread.currentThread().interrupt();
    System.out.println(Thread.interrupted()); // true
    System.out.println(Thread.interrupted()); // false
