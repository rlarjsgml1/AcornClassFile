서버와 클라이언트

서버 : 서비스르 제공하는 프로그램
서비스를 제공하는 프로그램 : 클라이언트


##########
ip: 컴퓨터 식별
port: 컴퓨터 내 프로그램 식별
##########

IP주소 : 네트워크에 연결된 컴퓨터의  고유의 주소
cmd창에서

ipconfig   로 확인가능하다

port : 하나의 컴퓨터에서 실행되는 프로그램을 구분함


IP:하나의 컴퓨터까지 찾아감
하나의 컴퓨터에는 여러개의 서버가 실행중이다 
원하는 서버프로그램을 선택하기 위해서는   port가 필요하다

네트워크프로그램 작성하기( tcp프로그램 작성하기 , 채팅프로그램)

Socket  : 서버컴퓨터와 클라이언트가 통신하기 위해서는 각 자 소켓이 필요하다
ServerSocket : 서버가 되기 위해서는 서버소켓 필요
               서버는 포트를 열고 대기에 빠진다
               요청한 클라이언트와 통신할 소켓을 생성함
