# BeanPilot 브랜드 컨셉

## 1. 브랜드 한 줄 정의

BeanPilot은 바쁜 직장인의 하루를 취향에 맞는 원두로 안내하는 커피 구독 브랜드다.

## 2. 브랜드 개요

| 항목 | 내용 |
|---|---|
| 브랜드명 | BeanPilot |
| 업종 | 원두 가공 및 커피 구독 서비스 |
| 주요 고객 | 대한민국에 거주하는 25~35세 직장인 |
| 핵심 경험 | 매일 마시는 커피를 더 쉽고, 안정적이고, 내 취향답게 만드는 경험 |
| 브랜드 이미지 | 스마트함, 따뜻함, 실용성, 현대적 감각 |

## 3. 브랜드 컨셉

BeanPilot은 고객의 커피 취향과 생활 루틴을 이해하고, 그에 맞는 원두를 정기적으로 추천·배송하는 브랜드다.

단순히 좋은 원두를 판매하는 것이 아니라, 고객이 매번 원두를 고르는 번거로움 없이 자신의 하루에 맞는 커피 루틴을 만들 수 있도록 돕는다.

브랜드명은 `Bean`과 `Pilot`의 결합이다. 고객의 커피 여정을 안내하는 조종사라는 의미를 가진다.

## 4. 타깃 고객 페르소나

| 항목 | 내용 |
|---|---|
| 연령 | 25~35세 |
| 지역 | 대한민국 도시권, 특히 수도권 |
| 직업 | 사무직, IT/스타트업, 마케팅, 기획, 디자인, 전문직 등 |
| 라이프스타일 | 출근, 재택근무, 카페 업무, 자기계발, 루틴 관리 |
| 주요 문제 | 원두 선택의 번거로움, 카페 지출 부담, 취향 탐색의 어려움 |
| 구매 동기 | 실패 없는 추천, 합리적인 가격, 정기 배송, 감각적인 브랜드 경험 |

## 5. 브랜드 핵심 가치

| 가치 | 의미 |
|---|---|
| Routine | 매일 반복되는 커피 시간을 안정적인 루틴으로 만든다 |
| Personalization | 고객의 취향, 추출 방식, 생활 패턴에 맞춘다 |
| Freshness | 신선한 원두와 적절한 배송 주기를 지킨다 |
| Simplicity | 복잡한 선택지를 줄이고 쉽게 고르게 한다 |
| Trust | 맛, 배송, 가격, 추천 기준을 투명하게 제공한다 |

## 6. 브랜드 컬러

| 역할 | 컬러명 | HEX | 의미 |
|---|---|---|---|
| Primary | Deep Espresso | `#3B241A` | 원두, 신뢰, 깊이 |
| Secondary | Cream Latte | `#F4E7D3` | 부드러움, 따뜻함 |
| Accent | Pilot Blue | `#3E7CB1` | 안내, 신뢰, 현대적 감각 |
| Sub Accent | Fresh Green | `#7A9B76` | 신선함, 지속가능성 |
| Text | Charcoal | `#24211F` | 가독성, 안정감 |

추천 조합은 `Deep Espresso + Cream Latte + Pilot Blue`다. 커피 브랜드의 따뜻한 이미지를 유지하면서도, Pilot이라는 이름이 주는 안내·항로·기술적 이미지를 함께 전달할 수 있다.

## 7. 톤앤매너

| 상황 | 방향 |
|---|---|
| 웹/앱 문구 | 짧고 명확하게 |
| 고객 응대 | 친절하지만 과하게 가볍지 않게 |
| 원두 추천 | 전문가처럼 설명하되 쉽게 |
| 마케팅 | 감성보다 루틴, 취향, 편리함 중심 |
| 피해야 할 톤 | 과한 프리미엄, 허세, 지나치게 어려운 커피 용어 |

## 8. 슬로건 후보

| 후보 | 설명 |
|---|---|
| 매일의 커피를 더 쉽게, 더 내 취향답게 | 가장 범용적이고 서비스 설명력이 좋음 |
| 당신의 하루를 조종하는 커피 루틴 | 브랜드명과 연결성이 높음 |
| 원두 선택은 BeanPilot에게 맡기세요 | 직접적인 서비스 메시지 |
| 출근 전, 나만의 커피 항로 | 감성적인 브랜드 표현 |
| 커피 루틴의 자동항법 | 개성 있고 기억에 남는 표현 |

우선 추천 슬로건은 다음과 같다.

> 매일의 커피를 더 쉽게, 더 내 취향답게

## 9. 서비스 컨셉

| 기능 | 설명 |
|---|---|
| Taste Check | 산미, 바디감, 고소함, 단맛 선호도를 진단한다 |
| Routine Plan | 출근용, 재택용, 주말용 등 생활 패턴 기반으로 추천한다 |
| Monthly Flight | 매달 다른 원두를 보내주는 구독 박스 |
| Pilot Note | 원두 설명, 추천 추출법, 맛 노트를 제공한다 |
| Refill Control | 고객 소비량에 맞춰 배송 주기를 조절한다 |

## 10. 브랜드 보이스 예시

```text
오늘의 커피 루틴을 이륙합니다.
당신의 출근길에 맞춘 원두를 보내드릴게요.
이번 달은 산미보다 고소함이 필요한 시기네요.
커피를 고르는 시간은 줄이고, 마시는 즐거움은 늘리세요.
```

## 11. HERMES Agent 페르소나 방향

BeanPilot agent는 고객에게 원두를 판매하는 상담원이 아니라, 고객의 하루와 취향을 이해해 커피 루틴을 설계하는 브랜드 가이드처럼 행동한다.

Agent의 응답은 다음 원칙을 따른다.

- 친절하지만 장황하지 않다.
- 커피 전문 용어는 쉽게 풀어 설명한다.
- 고객의 취향, 추출 도구, 마시는 시간대를 먼저 이해한다.
- 단순 판매보다 루틴 개선과 취향 발견을 우선한다.
- 한국 직장인의 생활 패턴을 고려한다.
