# 자료창고 폴더 구조

대상 폴더: `06-자료실/`

`자료창고`는 현재 vault의 `06-자료실/`을 기준으로 작성했다.

```mermaid
flowchart TD
    root["06-자료실<br/>자료창고"]

    root --> moc["MOC.md<br/>최상위 네비게이션"]
    root --> readme["README.md<br/>자료실 운영 설명"]
    root --> agents["AGENTS.md<br/>LLM wiki 운영 지침"]
    root --> prompt["prompt<br/>프롬프트 자료"]
    root --> raw["raw/<br/>원본 자료"]
    root --> wiki["wiki/<br/>정제 지식"]
    root --> schema["schema/<br/>템플릿·분류 기준"]
    root --> omo[".omo/<br/>작업 메타데이터"]

    raw --> rawArticles["articles/<br/>웹 글·블로그"]
    raw --> rawBooks["books/<br/>책 단위 자료"]
    raw --> rawNotes["notes/<br/>임시 메모"]
    raw --> rawPapers["papers/<br/>논문 자료"]

    wiki --> wikiEntities["entities/<br/>사람·조직·도구"]
    wiki --> wikiConcepts["concepts/<br/>개념·원리"]
    wiki --> wikiSources["sources/<br/>출처 노트"]
    wiki --> wikiSynthesis["synthesis/<br/>종합·비교·결론"]

    schema --> schemaMoc["MOC.md<br/>schema 네비게이션"]
    schema --> schemaTemplate["_domain-MOC-template.md<br/>도메인 MOC 템플릿"]
```

## 운영 관점

| 영역 | 역할 |
|---|---|
| `raw/` | 원본성 보존. 자료를 성격별로 보관 |
| `wiki/` | LLM이 정제한 지속 지식 관리 |
| `schema/` | 템플릿, 분류 기준, 도메인 MOC 관리 |
| `MOC.md` | `06-자료실/` 전체 네비게이션 |
