# Sources MOC

원본 자료를 LLM wiki에 통합하기 위한 정제된 출처 노트를 관리합니다.

## 페이지 목록

- 아직 등록된 source가 없습니다.

## 작성 기준

- 새 페이지는 [[wiki/sources/_source-template|Source Template]]을 사용합니다.
- 원본 위치는 `raw_path` 또는 `external_url`에 기록합니다.
- 출처 노트는 요약만 남기지 않고, 어떤 wiki 페이지를 갱신했는지도 기록합니다.
